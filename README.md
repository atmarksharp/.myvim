# .myvim

構造化した `.vimrc` .... ***"vimrc Anywhere"***

### Packages

- NeoBundle
- Emmet
- NERDTree
- Powerline
- vim-atom-dark (colorscheme)
- set tabwidth 2
- key-remap
- etc

### Install

```
cd ~ && git clone https://github.com/atmarksharp/.myvim.git
cd ~/.myvim && rake apply

# open vim, and type :NeoBundleInstall
```
