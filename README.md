# .myvim

構造化した `.vimrc` .... ***"vimrc Anywhere"***

### Packages

- NeoBundle
- Emmet
- NERDTree
- Powerline
- vim-atom-dark (colorscheme)

#### Others

- set *tabwidth* 2
- set *softtabstop*
- key-remap

### Install

```
git clone https://github.com/atmarksharp/.myvim.git ~/.myvim
cd ~/.myvim && rake apply

# open vim, and type :NeoBundleInstall
```
