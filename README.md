# .myvim

my .vimrc anywhere

### Packages

- NeoBundle
- Emmet
- NERDTree
- lightline
- vim-atom-dark (colorscheme)

#### Others

- set *tabwidth* 2
- set *softtabstop*
- key-remap

### Install

```bash
rm -rI ~/.vim/ # if you already installed plugins

git clone https://github.com/atmarksharp/.myvim.git ~/.myvim
~/.myvim/install.sh

bash ~/.myvim/install.sh
```
